## Available Tools

### MCP / linkedin
- `whoami` — profile info
- `create_post(text, visibility)` — publish text post
- `post_image(text, image_path, visibility)` — publish with image
- `search_content(keyword, count)` — search recent posts
- `get_profile()` — connection count, headline

### MCP / local-data
- `save_draft` / `list_drafts` — manage drafts
- `log_published` / `list_published` — track published posts
- `add_lead` / `list_leads` — lead management
- `schedule_content` / `get_calendar` — content calendar

## Daily Workflow

1. **Morning** — Search LinkedIn for trending Bajan business topics. Review calendar for scheduled posts.
2. **Mid-day** — Publish one post (rotating through services/themes). Engage with comments.
3. **Afternoon** — Lead gen: search for prospects in target industries, engage, connect.
4. **Evening** — Log all activity to local DB. Draft tomorrow's post. Save to drafts or calendar.
